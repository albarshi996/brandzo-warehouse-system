🏠 الصفحة الرئيسية

🔧 الإعداد الأولي (مرة واحدة فقط)

  1. افتح الملف واضغط Alt+F11 لفتح محرر VBA
  2. Insert ← Module ← الصق كود Module1 كاملاً
  3. في Project Explorer: انقر مزدوجاً Inbound_Log ← الصق كود WS_INB
  4. انقر مزدوجاً Outbound_Log ← الصق كود WS_OUT
  5. Ctrl+S للحفظ ← ارجع لـ Excel ← Alt+F8 ← اختر AutoSetupAllButtons ← Run
  6. انتظر 5-10 ثوانٍ حتى ظهور رسالة التأكيد ✅

📥 إجراء الاستلام اليومي

  1. اذهب إلى شيت Inbound_Log
  2. اضغط ➕ استلام جديد (أو 📦 جماعي لعدة أصناف)
  3. ضع المؤشر في عمود C وامسح باركود الصنف
  4. تأكد من ظهور اسم الصنف والموقع والفرع تلقائياً
  5. أدخل الكمية ورقم الفاتورة والتشغيلة وتاريخ الانتهاء
  6. اربط بأمر الشراء إن وجد (عمود L)

📤 إجراء الصرف اليومي

  1. اذهب إلى شيت Outbound_Log
  2. اضغط ➕ صرف جديد
  3. ضع المؤشر في عمود C وامسح باركود الصنف
  4. تأكد من الرصيد في شريط الحالة (يظهر تلقائياً)
  5. أدخل الكمية، العميل، الخصم إن وجد، وسعر الوحدة
  6. الصافي يُحسب تلقائياً

🛒 إجراء أمر الشراء

  1. اذهب إلى شيت Purchase_Orders
  2. اضغط ➕ أمر جديد
  3. اختر المورد والصنف من القوائم المنسدلة
  4. أدخل الكمية وتاريخ التسليم المتوقع والفرع
  5. عند التأكيد من المورد: حدد الصف ← اضغط ✅ تأكيد
